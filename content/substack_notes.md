## Note 2（2026年3月13日）

「我是那隻醬！板！鴨！」

早上被廣告打到，下意識要封鎖的時候發現哇好像是全新的東西==

封鎖了一堆中國小說網站廣告，封鎖了一堆中國短劇廣告，直到剛剛被推播了異世界轉生世界觀的動畫短劇（應該還是真人配音但畫面絕對 AI）。還知道前5秒的hook剪輯了擦邊的乙遊感==可以這麼色喔。

是我淺薄了，大模型老師。

-

就模式上，我認為選擇這個「魂穿、異世界、乙女向」賽道真的沒什麼破綻，畢竟用ai 輕易能做成更完整的世界觀跟美術設定，直接取代那些復仇轉生跟霸總類型化直立式短劇。（去年已批評過對我來說都是垃圾就不贅述）

我一直都覺得這種大量複製的類型本來就是就是 Porn，走在這個賽道上，你根本就不需要強勢 IP。不免又要拿 A 片出來舉例，每種喜好都會濃縮成標籤，每個場景甚至那個游泳池也會變成符碼，每個標籤交織再交織又會形成新的符碼，你不會知道，也許有人的性慾啟動點是那個游泳池。

-

樂觀一點來說，「用 AI 產出的異世界文體內容取代短劇垃圾」這我倒是樂見其成，畢竟短劇生態是很難養出明星的，但二次元可以輕易產出不同類型的美男。

資本看得到短劇的商業模式，他的商業模式是：所有生產環節都可以抽換，且即用即丟。很有效率，資本也喜歡效率，但無法為產業累積任何東西。我不想看到任何產業人員被粗製濫造的短劇耗損。之前還能看小紅書的時候，我經常看很多從業人員的小紅書。

後來理解短劇產業，我總覺得那就是大型的 MMORPG，劇場本來就是在扮演，短劇就是在扮演扮演本身，反正我也覺得所有的宮鬥古裝劇都是甄嬛傳和夢回大清的同人本，只是我覺得。

創作沒有靈魂，沒有想講的話，那全都是泡沫。

-

反正，對於這個我不知道該怎麼描述的產品：

可以想像，格式上，webtoon的類型故事可以直接轉成動畫短劇，畢竟篇幅、敘事結構都會很類似。

可以想像：內容上，小說網的明朝穿越架空小說可以更輕易的轉化成短劇或長劇的形式。當然，金庸也可以。如同標題寫到的醬板鴨，最近一堆醬板鴨迷因超好笑。

可以想像：遊戲的世界觀越完整，衍生的作品也會更豐富。

-

講結論，我認為除非成為頂尖的創作者，其他的都會變成算力的燃料（甚至不是算力本身，算力本身在我腦中是一堆搬運101011011的小人）。

我並不會感到悲觀，因為既有的產業環境本來有太多消耗人的事。換個形式一樣能「表達」，那沒有什麼不好。我們能快步飛越打造工具與消耗人類的過程，直接進入打造世界觀、宇宙觀。

本來人和人相遇就是兩個宇宙的相遇。

我們穿越人群遇見真心，也穿越演算法遇見真心。

-

我並不主張所有的影視製程都即將被 AI 「取代」。

Netflix 內容長 Bela Bajaria ：「我們相信，新工具應該擴展創作自由，而不是限制創作自由，或取代編劇、導演、演員與製作團隊的工作。」—— 引自《科技新報》，（收購華納兄弟失敗後，Netflix 轉向 AI：買下 Ben Affleck 創辦的電影技術公司）

至於劇場和現場演出一定會越來越興盛，因為劇場和現場都充滿限制，這我倒是覺得也充滿期待，套句宇宙玉說的「很會唱的人很多，但歌聲有故事的人很少」，抖抖的不安定感。

也許有一天抖抖的不安定感，演算法也算得出來，但是那個沒有意義。

抖抖的原因只有自己知道為什麼。

作為閱聽者，我還是先去運動好了，離開手機電腦遠一點。

## Note 26（2025年8月28日）

Ego is toxic

這幾天在用 gemini cli 寫一個一直很想做的分析邏輯，等他自己寫程式的過程有種難以言喻的感覺，隨手用文字追趕一下：所有解決方案都會過期，未來一直來。

*

有一次突然 panic 起來跑去問我爸：「貨幣會波動、資產會貶值，物質會消失，可能因為時局或市場或戰爭，任何累積任何時候都有可能歸零，如此一來累積還有意義嗎？怎麼辦？」

（會問這題應該是2021看到台幣匯率27元，當時加密貨幣各種牛，但我對貨幣的概念也是一直很模糊就是了）

父：（指了自己的頭） 只有腦袋裡的東西不會消失。

我：我只是想知道有沒有不會貶值的東西。

父：大部分的東西都只有交易的當下才有價值。只有你腦中的知識會留下來。

我：我會失智。

父：我們沒有失智的基因。

我：我會死。

我們都會死，我只想知道一堆人在買黃金是什麼意思、大家都在買加密貨幣是什麼意思。

黃金也有可能貶值啊，到時候是要回去用貝殼？

*

後來我想，知識有很多種型態，有時是記憶，有時是框架，有時是操作手冊，包裝成解決方案的形式出現。

到底在寫什麼東西突然有這種啟發，其實也很簡單，就只是想用自然語言去問資料庫。我知道 MCP 能做到，很多人都能做到，但我就是無法從 MCP 得到我想要的結果。

一開始是一個 prompt + 一個 loop，發現一直沒有辦法得到更細緻的答案，才分層，變成三個 prompt：

planner prompt：分析要寫幾次 SQL

SQL generator：寫完去跑 SQL

summary generator：根據知識和計算結果總結

每次寫出來都覺得一點魔法也沒有，但我從1 到 3的過程感受到了新的思路。

*

未來一直來，當解決方案成為歷史 WE MUST COMPACT THE CONTEXT

當中的每一部分進入集體意識，思想會在聚散之間產生。

確切感受到水瓶時代的能量，我們要跟著流動走，流動必先臣服，ego 阻礙臣服。

## Note 28（2025年7月1日）

#我對愛情的貧乏想像

在卡米地遇到追蹤者來相認，促使我也重聽了前陣子上阿部社長的會議室 podcast 討論到愛情的段落。那段我回答得不好，因為我並沒有 100% 誠實，大睡了一覺後決定來自由書寫。

同時也想到，以前上過加恩的 podcast，閒聊時激進主張婚姻是兩人合夥開一間公司。我一開始只是想比喻談論到簽約合作，不外乎就是講清楚雙方期待。想要長遠的關係，本來就可以將協議視為一個理性的工具，如此婚前協議十分可行。

總希望有人可以反駁我，但沒有，所以我要來反駁自己

--

協議只能提供物質生活的框架，包含但不限於財產分配到誰不洗碗誰洗衣服。我一直覺得，看一份合約設計，可以看出擬約的人對未來的想像有多遠。

關係是動態的，個體的養成也是。

也許這就是智能合約辦不到的事，人性不變，但人心會變，所以智能合約理想中的理性世界很難成形。

--

我總是在嘲笑「永遠能有多遠」，若要對自己百分之百誠實，其實我相信永遠。

朋友說：「我不 commit 但我一 commit 就是買房。」

我為對感情展露出玩世不恭的態度感到羞赧。

--

當我從他人的婚姻中看到物質上的安全感，宇宙就把這樣的對象體驗卡放到我面前；我開始意識到我「絕對不要」，宇宙馬上就會發出紅牌，讓拖拖拉拉的我做出決定。

由衷感謝被照顧得像個公主的時候。

--

是要生活的夥伴，還是實踐浪漫愛？

也許我（看似）激進的主張愛、婚、生、育都可以脫鉤，其實我真正想得這個問題的答案：

婚的問題排除後，如果我足夠強大到能獨立面對生和育，這樣就能更接近愛嗎？

（至於婚，我只能說，太喜歡參加婚禮了——儀式我愛，制度我還是打一個問號。）

## Note 50（2025年3月25日）

（音樂劇產業觀察筆記）

政府的支持和補助很重要。

我也曾經抱有「供過於求的問題應由市場機制自然調整」的心態。

隨著更加深入產業，我的心態開始發生轉變。

市場是理性的，資本是無情的，如果沒有底線，任憑市場決定文化的走向，文化多樣性勢必會收到威脅，畢竟每種形式的傳播條件都不同。

推行喜劇和音樂劇，對我來說有共同的策略，是雅俗共賞的「俗」。更多未曾進劇場的人進到劇場，也許就讓更多人有觀看現代舞的理由。

期待更多表演團隊公司化，形成小小的艦隊。

-

最後，很感謝 榮耀基金會 GLORY Foundation for Creative Industries 積極行動，短短八個月內，召集演出團隊、業界師資、核心觀眾，最後一天更是動員了近百人進行圓桌會議，以及超過百人參與的提案大會。這麼多場次、賓客的進出場也好複雜，毛毛 Michaelia Sun bravo！

以能給出的資源和現金來說，和氣資本既沒有場地、也沒有售票系統，是小小小投資者，非常榮幸能和其他業界先進一起參與計畫，我的收穫非常多。

雷爸說：「看現在的音樂劇集結，好像幾十年前看台灣資訊業。」

在市場衝鋒陷陣過的前輩，無私又大器的提供資源和經驗，更登高一呼提供獎金，有這樣的開始感覺未來充滿希望。產業化的路上，數位化、流程化、專業分工勢在必行，在有型的產業聚落生成之前，我們先相信—

如果南韓花了30年的時間讓原創音樂劇開始走向世界，台灣的三十年現在開始也可以，讓好故事深入人心。

團結！
